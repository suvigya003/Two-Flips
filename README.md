# Two Flips - A Brain Teaser Game
## [See Live](https://two-flips.vercel.app/) 
Two Flips is a game which is a memory exercise game. The user needs to remember the images on all cards before they flip. Then the test begins where user has to match the cards with similar images else it is a fail.
### Technologies used:- 
- React
- JavaScript
- HTML
- CSS
